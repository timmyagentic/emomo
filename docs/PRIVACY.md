# emomo Privacy Policy

Last updated: 2026-06-03

emomo is a no-login meme search app. This policy describes the data handled by the mobile app and backend service.

## Data We Process

- Search queries: When you search, the query text is sent to the emomo backend so the service can return matching meme images.
- Technical request data: The backend may process basic request metadata such as IP address, timestamp, route, response status, and user agent for security, abuse prevention, debugging, and service reliability.
- Local search history: Recent searches are stored only on your device through local storage. You can clear this history in the app.
- Photo library access: emomo requests photo-library permission only when you choose to save a meme image.

## Data We Do Not Collect

- emomo v1 does not include user accounts.
- emomo does not sync search history across devices.
- emomo does not access contacts, location, camera, microphone, or push notifications.
- emomo does not sell personal data.
- emomo does not use third-party advertising tracking SDKs.

## Image Sharing And Saving

Sharing uses the system share sheet. Saving writes the selected meme image to your photo library after permission is granted. Copying a link writes the image URL to your clipboard.

## Retention

Search history stored in the app remains on your device until you clear it or uninstall the app. Backend operational logs, if retained, are used for service reliability and abuse prevention.
